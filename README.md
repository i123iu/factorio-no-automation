# Factorio: No Automation

A mod for Factorio that removes all forms of automation - craft items by hand, mine ores manually and drive trains. No robots!

Space Age is not supported. 

## Gameplay Changes

Changed:

- Assembling machines can now only craft recipes that require fluids
- Electric mining drills only work on uranium ore (and are unlocked through uranium mining technology)

Removed:

- Burner mining drills
- Train stops and signals
- Construction and logistic robots
- Roboports and personal roboports
